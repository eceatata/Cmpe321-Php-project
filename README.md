## Database project:

This is a simple database management system program. Its functions listed below:

    DDL Operations:
        1-> Create a type
        2-> Delete a type
        3-> List all types

    DML Operations:
        1-> Create a record
        2-> Delete a record
        3-> Search for a record
        4-> List all records of a type
